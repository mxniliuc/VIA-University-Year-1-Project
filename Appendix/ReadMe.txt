How to run:
1 - Extract "G4_SEP1Z_A20_IntelliJ_Project.zip" file
2 - Open IntelliJ Idea
3 - Click File
4 - Choose Open
5 - Navigate to the folder where you extracted the .zip file
6 - Select the project folder and click OK
7 - Expand the "G4_SEP1_ManagementSystem" module
8 - Expand its "src" folder
9 - Right click on Main class and choose "Run 'Main.main()'"

Extra:
If there are any issues regarding jdk, try to resolve those :D
